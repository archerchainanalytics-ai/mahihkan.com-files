On the Record — What This Is

AcerbE™ is not a DRM scheme with a new name. It is not a license server that phones home. It is not a soft warning system that logs events and sends emails. It is a forensic watermarking and integrity verification engine that embeds cryptographic fingerprints into software at the byte level, detects tamper events deterministically, and produces a chain-verifiable audit record that constitutes forensic evidence. It does not ask whether something happened. It records that it happened — with a timestamp, a hash, and a chain anchor. That record does not disappear. It does not negotiate. It is evidence.

Origin

Why This Exists

Software gets stolen. Code gets copied, rebranded, and resold. Licensing systems get bypassed. Developers spend months building something and watch it appear under a different name six weeks later with no recourse, because proving provenance in software has historically been nearly impossible without an expensive legal process that the other side can simply delay.

AcerbE™ was built to end that. It embeds a forensic fingerprint into every product it protects at the point of delivery. That fingerprint is cryptographic, chain-anchored, and registered against the Archer Chain Analytics registry. If the software is tampered with, the fingerprint records it. If it is redistributed without authorization, the fingerprint identifies it. If it ends up in a dispute, the fingerprint is the evidence — not an assertion, not a claim, a verifiable cryptographic record.

It also powers the AcerbE™ FAFO Guarantee — our warranty void condition. If a customer tampers with an Archer Chain Analytics product and AcerbE™ has logged the event, the warranty is void. We will know. The record does not disappear.

Tamper Warning — Read Before Purchase

AcerbE™ is active in every Archer Chain Analytics product it protects. It does not announce itself during normal operation. If you attempt to modify, reverse-engineer, bypass licensing, or otherwise interfere with the integrity of a protected product, AcerbE™ will log the event. The log is timestamped, hashed, and chain-anchored. We will have a forensic record before you know it has fired. This record voids the AcerbE™ FAFO Guarantee and may be presented as evidence in any dispute. There is no appeal to the fingerprint. There is no patch for the log.
